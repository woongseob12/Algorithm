# 특징
- LIFO(Last In First Out)

# 사용법

```C++
#include <stack>    
stack<int> s;      
```

# 기타 Method

```C++
stack.push(1);       
stack.top();        
stack.pop();       
stack.size();       
stack.empty();       
```
